# 🌸 Iris Flower Classification

A Machine Learning Classification project developed using Scikit-Learn.

## Features

- Data preprocessing
- Random Forest Classifier
- Model Evaluation
- Streamlit Web App
- Model Saving using Joblib

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Accuracy

~100%

## Run

```bash
pip install -r requirements.txt

python src/train.py

streamlit run app.py
```
